About

RT Proxy is a simple tool to proxy RT requests (see A standard for RT'ing?) to different social sites.

This page takes three possible query string parameters: title, link and description although Delicious is currently the only service that takes a description.

Demo

Fill out the following fields and click Test. It submits back to this page with the query string parameters filled out. Then you can click any of the links to the left to submit to that service.
